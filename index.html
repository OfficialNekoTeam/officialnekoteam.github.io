主题

NekoBot多平台智能聊天机器人框架

基于 Quart + asyncio 的事件驱动框架，支持多平台与 LLM 集成

快速开始
GitHub
Web 仪表盘
✨

多平台支持

支持 QQ (OneBot V11)、Telegram、Discord、飞书、KOOK、QQ频道、Slack、企业微信等 8 个主流聊天平台，轻松实现跨平台消息同步和管理。

🤖

多 LLM 集成

集成 OpenAI、Google Gemini、Claude、DeepSeek、DashScope、Moonshot、ZhipuAI 等 10+ 个主流 LLM 服务商，提供强大的对话能力。

🧩

插件系统

完善的插件生态，支持热加载、热重载，通过插件快速扩展功能，支持本地和在线插件。

📊

Web 仪表盘

基于 React + TypeScript + Vite + Material-UI 构建的现代化管理界面，提供友好的可视化操作体验。

🎯

Agent 系统

支持 MCP 协议、函数调用和工具管理，构建强大的 AI Agent 能力。

📚

知识库系统

支持向量数据库、文档解析和检索增强（RAG），实现智能知识管理和问答。

🔄

消息流水线

洋葱模型的消息处理管道，支持多种处理阶段，灵活定制消息处理流程。

⚡

高性能架构

基于 Quart (异步 Flask) 的异步架构，高性能处理消息和请求，支持实时日志推送。

🔒

安全可靠

JWT 认证、bcrypt 密码加密，确保数据和账户安全，支持会话隔离和权限管理。
